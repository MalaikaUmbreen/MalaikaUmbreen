## 🔬 Featured Research

### Network Steganography Detector — Final Year Project

> *Transformer-based deep learning model for detecting hidden covert channels in network traffic*

| Detail | Info |
|--------|------|
| **Method** | Vision Transformer (ViT) architecture adapted for packet flow classification |
| **Dataset** | Custom-labelled PCAP captures with steganographic payloads |
| **Detection Targets** | Covert channels in DNS, HTTP, and TCP header fields |
| **Stack** | Python · PyTorch · Scapy · Wireshark · Pandas · Scikit-learn |
| **Status** | 🔄 In progress — model training & evaluation phase |

**Why it matters:** Network steganography is one of the stealthiest APT exfiltration techniques — hiding malicious data inside innocent-looking traffic. Traditional signature-based SIEM rules cannot detect it. This research applies modern deep learning to close that gap.

📁 [View the repository →](https://github.com/MalaikaUmbreen/Network-Steganography)
